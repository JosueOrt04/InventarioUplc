# Sistema Web de Gestión de Inventarios de Laboratorio (UPLC)

Plataforma web desarrollada para la gestión integral de materiales y reactivos del
laboratorio ambiental de una universidad. El sistema reemplaza registros manuales
por una solución digital centralizada y segura.

## Funcionalidades
- Control de usuarios con roles y permisos
- Registro de entradas, salidas, préstamos y devoluciones
- Módulo de almacén para nuevos reactivos
- Sistema de reservas del laboratorio
- Autenticación segura con JWT

## Tecnologías utilizadas
- Frontend: React, Tailwind CSS, DaisyUI
- Backend: Node.js, Express
- Base de datos: MongoDB
- Autenticación: JSON Web Tokens (JWT)

## Rol personal
Análisis del problema, diseño de la arquitectura y desarrollo completo del frontend
y backend como solución funcional para una institución universitaria.
